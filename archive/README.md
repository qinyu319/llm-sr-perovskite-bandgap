# Historical Audit Trail

This directory preserves non-authoritative model-development and publication
build material. It documents how the final model was reached but must not
override the revised paper or frozen final evidence.

The authoritative analytical model is the eight-term CsCl-containing M4:

```text
Sn, Br, Cl, Cs, Sn^2, Br^2, Cs*Sn, Cs*Cl
```

Use `final_m4/` and `final_m4_diagnostics/` for final-model evidence.
`pruned_m4/` contains an earlier pruning interpretation, and
`archive/historical_model_stage_history.xlsx` records historical stage models.
