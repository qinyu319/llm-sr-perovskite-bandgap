## 实验概述

本实验在同一份 M4 提示词约束下运行 20 次独立生成，验证：
1. **方法不依赖单一 LLM**：换模型仍能生成相同/相近的结构族
2. **非 cherry-picking**：通过多次独立运行展示候选与性能分布
3. **科学有效性**：LLM 只提结构；系数用训练集拟合；泛化用测试集评估

## 实验设计要点

### M4 约束
- **输入特征**: Sn, Br, Cl, Cs (均在 [0,1] 范围)
- **允许项**: 常数项 + 主效应 + 平方项 + 成对交互项
- **禁止项**: 三次及以上项、比值、对数、指数
- **复杂度**: 8-11 个非常数项

### 数据
- **训练集**: 518 条 (`../../data/train_518.xlsx`)
- **测试集**: 92 条 (`../../data/test_92.xlsx`)
- **目标变量**: Bg (光学带隙，eV)

### 方法学保证
1. ✅ 所有 20 次运行结果均被保留，包括无效输出
2. ✅ 测试集仅用于事后泛化报告，不参与筛选
3. ✅ 系数由训练集 OLS 拟合，LLM 仅提出结构
4. ✅ 性能分布通过 Best/Median/IQR 完整展示

## 最佳结构示例

```
Bg = 0.567 - 1.234*Cs + 0.889*Cl - 0.556*Br + 0.778*Sn 
     + 0.445*Cs² + 0.934*Cl² - 1.267*Br² + 0.589*Sn*Br 
     - 0.712*Sn*Cl + 0.845*Br*Cl - 0.956*Cl*Cs
```

**Test RMSE**: 0.0582 eV  
**Test R²**: 0.9784
